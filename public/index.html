URL Shortener

Instructions:

Pass a full url into the /new/ directory and get an object containing a new "shortened" URL
Paste shortened URL into address bar and you'll be redirected to the original site

Example:

https://tsmic.herokuapp.com/new/https://www.google.com

https://tsmic.herokuapp.com/1450166400

return example:

{"unixtime":1450166400,"natural":"December 15, 2015"}
