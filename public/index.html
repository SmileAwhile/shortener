URL Shortener

Instructions:

Pass a full url into the /new/ directory and get an object containing a new "shortened" URL
Paste shortened URL into address bar and you'll be redirected to the original site

Example creation:

https://shorterer.herokuapp.com/new/https://www.google.com

Example output:.

{ "_id": "57841267d8610a0011a4bee4", "original": "https://www.google.com", "shorter": "https://shorterer.herokuapp.com/57841267d8610a0011a4bee4" }

Usage example:

https://shorterer.herokuapp.com/57841267d8610a0011a4bee4

Redirected to example:

https://www.google.com
